js编写位置

添加标签的onclick属性中

<button onclick="alert('点什么啊')"></button>

##写在超链接的href中

<a href="javascript:alert('不要乱点我');">链接</a>

写在标签的属性中，属于结构与行为的耦合，不方便维护，尽量避免使用

正确操作：将js代码写到script标签中或者外部js文件

  1. 写在标签中
<script type="text/javascript">
    alert("你在干嘛");
    document.write("yy");
    console.log("有事吗");
    console.log("dd");
</script>
  1. 新建js文件，通过script标签引入js
    • 可以在不同的页面中同时引用，也可以利用到浏览器缓存机制
    • script标签一旦引入外部文件，就不能再编写代码了，浏览器会优先使用外部js文件，如有需要可以再次创建一个script标签
<script type="text/javascript" src="./demo.js"></script>

js字面量和变量

  1. js中严格区分大小写
  2. js每一条语句以分号结尾 ；不写分号浏览器会自动加分号，但是会消耗一些系统资源
  3. js会忽略多个空格和换行，可以利用空格和换行对代码进行格式化

  • 字面量：不可改变的值 1 2 3 4 5，可以直接使用，使用不方便

  • 变量：变量可以用来保存字面量，变量的值可以任意改变，更加方便使用

  • 可以通过变量对字面量进行描述var age=23

  • 开发中一般使用变量保存字面量

    声明变量

    使用var来声明变量

  var a;
  //声明以后的默认值是undefined
  a=123;
  alert(a);
  var b = 34;

js标识符

  • js中所有的可以自主命名的都可以称为标识符，例如：变量名，函数名，属性名都属于标识符

标识符命名规则

  • 标识符可以含有字母、数字、下划线、_ 、$(除此之外都不行)
  • 标识符不能以数字开头
  • 标识符不能是ES中的关键字或保留字
  • 标识符一般采用驼峰命名法：首字母大写，其余小写 HelloWorld
  • js底层保存标识符采用的是unicode编码，所以理论上讲，所有的utf-8中含有的内容都可以作为标识符

js字符串

数据类型指的是字面量的类型
在js中一共有6中数据类型

  • String 字符串
  • Number 数值
  • Boolean 布尔值
  • Null 空值
  • Undefined 未定义
  • Object 对象
    基本数据类型：String Number Boolean Null Undefined

js检查变量的类型

typeof 变量

字符串String中：

  • js字符串要用引号引起来，没有引号就是变量
  • 单双引号都可以，但不能混用
  • 引号不能嵌套，只有单引号里放双引号
  • 字符串中可以使用\作转义字符，"' \n换行 \t制表符

Number中：

  • 在js中所有的数值都是Number类型，包括整数和浮点数（小数）
  • 数字的最大值 Number.MAX_VALUE，使用Number表示的数字超过了最大值，则会返回Infinity，-Infinity（父无穷）
  • NAN 是一个特殊的数字 Not A Number，使用typeof检查NAN和Infinity都会返回Number
  • js中的最小值Number.MIN_VALUE
  • js中整数的运算基本上可以保证准确 但是 0.1+0.2 != 0.3

Boolean值：

  • true--真 false--假 做逻辑判断

Null类型的值

  • 用来表示为空的对象
  • typeof(Null)会返回Object

Undefined

  • 变量声明了但没赋值时

强制类型转换

String Number Boolean

转换为String

  • 调用被转换数据类型的toString()方法
  • null 和 undefined没有toString()方法
var a = 123;
var b = a.toString();
  • 调用String()函数
  • 对于Number和Boolean实际上就是调用的toString()方法(底层实现），但是对于null和undefined，不会调用toString()方法 。（String()直接把null转换成字符串"null"，undefined一样）
a=123
String(a)

转换为Number

  1. 调用Number函数（不能应对字符转）
Number(a)
  • 字符串--->数字
  • 如果是纯数字的字符串，则直接转换为数字
  • 如果字符串里有非数字内容，则转换为NAN
  • Boolean true --> 1 false --> 0
  • Null --> 0
  • Undefined -->数字NAN
  1. 当要转换的对象既有数字又有字符串时（123px）
  • parseInt() 把一个字符串转换为一个整数
a = "123px"
a = parseInt(a)
--> 123
a = "123.3456"
--> 123
a = "b123dmoa"
--> NAN
a = "123bwqw334"
--> 123
  • parseFloat() 把一个字符串转换为浮点数
  • 对非String使用parseInt()或者parseFloat()，会先转换成String再变成Number

其他进制的数字

  • 16进制的数字，需要以0X开头，输出仍然是十进制
a  = 0Xff
console.log(a)
--> 255
  • 8进制数，以0开头
  • 2进制数，0b开头但是不是所有浏览器都支持
  • 字符串"070" 有些浏览器会按照8进制解析，有些会按照10进制解析，解决办法
a ="070"
-->70
-->56
a = parseInt(a,10)//指定数字的进制

转换为Boolean值

  • 使用Boolean()函数
  • 数字 --> 布尔：除了0和NAN为false，其他都是true
  • 字符串-->布尔：除了空串为false，其他都是true
  • null和undefined都是false
  • 对象也会转换为true

算数运算符

运算符

通过运算符可以对一个或者多个值进行运算，并获取运算结果

  • typeof运算符，可以获得一个值的类型，并且以字符串的形式返回

算数运算符

  • + :对两个值进行加法运算，并将结果返回
  • 任何值和NAN做运算都是NAN
  • 对两个字符串进行+运算，进行拼串操作
  • - ： 对两个值进行减法运算并且进行返回
  • 任何值做减法运算时都会转换成Number，可以利用这一特点做隐式转换
a = "123"
var b = a - 0
//将a转换为Number

一元运算符（只需要一个操作数）

    • : 对数字没有影响
  • - : 可以对数字进行取反
  • * ：
  • /
  • =
