使用码云管理仓库

  1. 2023-04-06
  2. 161字
  3. 1 min read

添加公钥

  • 生成公钥（注意git环境变量问题）
  • 头像->设置->左边安全设置->ssh公钥
  • 点击查看怎样生成公钥，按照步骤来，生成公钥后去对应的文件夹下找到公钥复制添加

添加仓库

注意： 不要选设置模板

  • 创建成功后打开项目的PowerShell
  • git status 查看项目状态
  • git add . 提交所有文件
  • git commit -m "add files" 在本地提交一次项目
  • git remote add origin https://gitee.com/vsdeveloper/仓库名.git
  • git push -u origin master 将本地仓库和远程仓库关联
  • 成功上传到码云
