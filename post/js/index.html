js对象类型

  1. 2023-03-19
  2. 309字
  3. 2 min read

  • 基本数据类型是一个单一的值，值和值之间没有联系

  • 对象是一种复合式的数据类型，对象中可以保存多个不同数据类型的属性

    对象的分类

  1. 内建对象：
  • 由ES标准中定义的对象，在任何的ES的实现中都可以使用，例如：MATH String Number
  1. 宿主对象
  • 由JS的运行环境提供的对象，例如BOM、DOM
  1. 自定义对象
  • 开发人员自主创建的对象

自定义对象

创建自定义对象

  1. 使用new关键字调用的函数，是构造函数的constructer
    • 构造函数是专门用来创建对象的函数
var object = new Object();

在对象中添加属性

  • 向对象添加属性
  • 语法：对象.属性名 = 属性
//向obj中添加name属性值
obj.name = "孙悟空"；

读取对象中的属性

console.log(obj.name);

读取对象中没有的属性会返回undefined

修改属性值

对象.属性名 = 新值

删除对象属性

delete obj.name

属性名

  • 对象的属性名不强制要求遵守标识符的规范
  • 如果要使用特殊的变量名，需要使用如下语法：
    obj["123"] = 789;
